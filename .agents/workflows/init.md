---
description: Scans the codebase, analyzes its architecture, and automatically creates standard documentation (like ANTIGRAVITY.md) and a persistent Knowledge Item (KI) to shortcut future context loading.
---

1. Use `list_dir` or `view_file` to review `package.json`, `README.md`, or the primary `src` structure to understand what this project is.
2. Draft an `ANTIGRAVITY.md` file capturing the core architecture boundaries, project structure, and important developer commands.
3. Automatically generate a robust Knowledge Item (KI) metadata & architecture file inside `<appDataDir>\knowledge\[project-name]` summarizing your findings, so that future coding sessions load this knowledge instantly via KI Summaries.
4. Tell the user what was created and that the repository has been fully indexed.
